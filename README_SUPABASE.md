# 旅行助手 - Supabase云端集成版

## 🎉 项目简介

旅行助手应用已成功集成Supabase后端，实现用户认证、设置云端同步和背景图片云存储功能。

- ✅ 保持原有UI设计（零变化）

## 🛠️ 本地开发

### 环境要求
- Node.js 18+
- pnpm 8+

### 安装和运行
```bash
# 进入项目目录
cd /workspace/travel-agent-ui

# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 构建生产版本
pnpm run build

# 预览生产版本
pnpm run preview
```

## 🔐 技术栈

### 前端
- React 18.3
- TypeScript 5.6
- Vite 6.2
- TailwindCSS 3.4
- Framer Motion 12
- React Router 6.30

### 后端
- Supabase (认证、数据库、Storage)
- PostgreSQL (数据存储)
- Edge Functions (无服务器函数)

### 关键依赖
- @supabase/supabase-js 2.80.0
- lucide-react 0.364.0 (图标)
